please don't use obscure acronyms. just use the words.

Acronyms look like they save a few seconds, but they often cost everyone minutes.

Not everyone works in your department. Not everyone works in your industry. Not everyone interprets the same acronym the same way.

❌ Don't do this

Alex 2:15 PM

Can you send the PM the PRD by EOD?

Sam 2:16 PM

PRD??

Alex 2:16 PM

Product Requirements Document.

Sam 2:17 PM

PM....... ?

Alex 2:17 PM

Project Manager.

Sam 2:18 PM

What's EOD?

Alex 2:18 PM

End of day.

Sam 2:19 PM

I'll send it to them by 4 PM.

Alex tried to save a few seconds by using acronyms here, but notice how much longer it took to convey the intent to Sam?

Some acronyms have multiple meanings.

PM can mean different things. Sam isn't sure who Alex wants him to send the document to:
Project Manager
Product Manager
Prime Minister

Just use the words 😩

✅ Instead try this

Alex 2:15 PM

Can you send the project manager the product requirements document by the end of the day?

Sam 2:15 PM

I'll send it to them by 4 PM.

The intent was conveyed in a fraction of the time, and Sam was able to start working right away without any ambiguity about the ask, or who to send it to.

A simple rule

  • If the reader might not know the acronym — write it out.
  • If you aren't sure — write it out.
  • If it's the first time you're mentioning it — write it out.
  • If typing the full phrase takes less time than explaining it later — definitely write it out.

What about the obvious ones?

Everyone knows what USA, FBI, and NASA mean. Use those. The test is simple: would a new hire, a contractor, or someone from another team know it without looking it up?

If you have to ask "should I define this?", define it. Everyone will thank you, including your future self.

The internet has enough confusion already.

Use words.
