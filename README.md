# noacronym.com

A simple static website inspired by nohello.net

The goal is straightforward: encourage people to communicate clearly by avoiding unnecessary acronyms and abbreviations that can confuse readers, especially across teams, industries, cultures, and languages.

The site is intentionally lightweight and built with plain HTML, CSS, and JavaScript. No frameworks, build systems, or backend services are required.

## Project Structure

```
├── assets/
│   └── favicon/
├── index.html
├── styles.css
├── script.js
└── README.md
```

## Development Philosophy

This project favors simplicity over functionality.

The current implementation is intentionally minimal and designed to be hosted directly on GitHub Pages without any build process.

As the project evolves, improvements will be added incrementally. The goal is to make small, thoughtful changes over time rather than redesigning the site all at once.

Future planned improvements include:

* Add profile pictures for the chat example
* Localization support

## Running Locally

Clone the repository and open `index.html` in a browser.

No dependencies, package managers, or installation steps are required.
